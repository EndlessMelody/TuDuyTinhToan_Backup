# TuDuyTinhToan_Backup (Computational Thinking Backup)

## Project Overview
This project focuses on Computational Thinking ("Tư Duy Tính Toán"). It serves as the primary repository for managing the learning, teaching, and experimental aspects of this domain.

## Getting Started
To get started, follow these steps:
1. Run the `01-project-init.md` project initialization workflow via the agent if not already initialized.
2. Review the `.cursorrules` to understand the coding guidelines and AI agent behaviors.
3. Review `.agents/workflows` to understand how to leverage automated steps for testing and deployment.

## Agent Team Setup
This repository uses the Agent Team Setup pattern to enforce developer productivity, standardize workflows, and integrate seamlessly with AI coding assistants. See `.cursorrules` and `.agents/workflows/` for more information.
